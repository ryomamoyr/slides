Marpによるスライド作成

概要

環境
VScode + Marpによるマークダウンベースのスライド作成
テンプレート
template_project/template_event/template.mdをコピーして使用
管理方法
プロジェクト毎・報告会毎に整理されたフォルダ構造

フォルダ構造

ディレクトリ構成イメージ
project_name/ ├── yyyyMMdd_eventA/ │ ├── images/ // スライド用画像 │ └── yyyyMMdd_報告資料.md └── yyyyMMdd_eventB/ ├── images/ // スライド用画像 └── yyyyMMdd_報告資料.md

各プロジェクトディレクトリには報告会などのイベントディレクトリが含まれ、その中にスライド（.mdファイル）とスライドに使う画像を保存するディレクトリが配置されています。

実例
A社/ └── yyyyMMdd_中間報告会/ ├── images/ └── 報告資料.md

スライドデザイン

theme.cssファイルで細かい設定を変更できます。以下の3つが定義されています

1
タイトルスライド
プレゼンテーションの表紙として使用するデザイン
2
目次スライド
プレゼンテーションの構成を示すレイアウト
3
章区切りスライド
セクションの変わり目を視覚的に表示するデザイン
